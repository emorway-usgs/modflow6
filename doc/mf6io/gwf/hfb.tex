Input to the Hydraulic Flow Barrier (HFB) Package is read from the file that has type ``HFB6'' in the Name File.  Only one HFB Package can be specified for a GWF model.  This Package previously allowed only barriers to horizontal flow and was called the Horizontal Flow Barrier Package.  It now also allows barriers to vertical flow, a capability similar to that in MODFLOW-USG \citep{modflowusg}.
